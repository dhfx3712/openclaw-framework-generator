## 首次启动流程

```
Step 1: 确认配置  →  检查 config.json 和 config.schema.json
Step 2: 连接底座  →  验证数据底座可正常读写
Step 3: 测试脚本  →  逐个运行 scripts/ 下的脚本验证
Step 4: 注册 cron →  注册定时任务
Step 5: 验收指令  →  逐条测试 TOOLS.md 中的指令
Step 6: 正式使用  →  开始录入数据
```
